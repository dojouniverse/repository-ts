# @dojouniverse/repository-ts

[![License](https://img.shields.io)](https://opensource.org)
[![Website](https://img.shields.io)](https://dojouniverse.com)

Official TypeScript repository patterns for the **DojoUniverse** ecosystem. Built with a focus on **Clean Architecture**, testability, and enterprise-grade scalability.

## Overview

`@dojouniverse/repository-ts` provides a language-agnostic architectural foundation for the Repository Pattern. Part of the DojoUniverse standard library, this package ensures consistent data access layers across TypeScript projects.

## 📦 Installation

```bash
npm install @dojouniverse/repository-ts
```
